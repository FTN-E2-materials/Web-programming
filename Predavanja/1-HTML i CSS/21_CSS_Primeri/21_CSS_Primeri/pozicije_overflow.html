"Overflow" svojstvo odredjuje sta da se uradi ukoliko sadrzaj elementa prelazi zadate granice visine i sirine.

Mozete menjato "overflow" svojstvo kako biste imali bolju kontrolu prikaza. Pokusajte da promenite ovo svojstvo na: visible, hidden, auto, or inherit i vidite sta se desava. Predefinisana vrednost je "visible".
