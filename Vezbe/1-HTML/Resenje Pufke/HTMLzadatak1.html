Web čitači

Web čitač je aplikacija koja je namenjena prikazu sadržaja web stranica.
Stranice se mogu dobaviti kako iz fajl-sistema, tako i putem TCP/IP mreze.
Protokol kojim komuniciraju web citaci i web server je HTTP protokol. Danas su najpoznatiji sledeci web citaci:

  1. Microsoft Internet Explorer (http://www.microsoft.com/ie)
  2. Mozila (http://www.mozilla.org)
  3. Opera (http://www.opera.com)

Obratite paznju da su svi nazivi web citaca u gornjoj listi linkovi koji upucuju na adrese navedene u zagradi!
